# Itinerario Cono Sur & Fin del Mundo 🌍

Página web del itinerario de viaje grupal - Marzo 2026.

## Despliegue local

```bash
npm install
npm start
```

Abrir en: http://localhost:3000

## Despliegue en Railway

1. Subir este repo a GitHub
2. En Railway: New Project → Deploy from GitHub repo
3. Railway detecta Node.js automáticamente
4. La variable `PORT` la asigna Railway sola
